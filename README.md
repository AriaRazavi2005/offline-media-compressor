# ⚡ Offline Media Compressor v2.0 (کمپرسور آفلاین مدیا)

یک ابزار وب کاملاً آفلاین، امن و بسیار سریع برای فشرده‌سازی ویدیو و تصویر در مرورگر کاربر. این پروژه بدون نیاز به ارسال حتی ۱ بایت اطلاعات یا فایل به سرور‌های خارجی کار می‌کند و حریم خصوصی کاربران را ۱۰۰٪ تضمین می‌کند.

---

## ✨ ویژگی‌های جدید و کلیدی در نسخه ۲.۰

- **🔒 ۱۰۰٪ آفلاین و حامی حریم خصوصی:** تمام پردازش‌ها (انکود ویدیو و تصویر) مستقیماً روی مرورگر دستگاه کاربر انجام می‌شود.
- **🔄 فشرده‌سازی دسته‌ای تصاویر (Batch Compression):** انتخاب و فشرده‌سازی چندین فایل به صورت همزمان و دانلود نهایی همه در یک فایل ZIP.
- **🖼️ پشتیبانی از فرمت فوق مدرن AVIF:** علاوه بر WebP و JPEG، حالا می‌توانید تصاویر خود را به فرمت مدرن **AVIF** با حجم بسیار کمتر تبدیل کنید.
- **🎬 فشرده‌سازی ویدیو با امکان بی‌صدا کردن (Mute):** قابلیت فشرده‌سازی ویدیو به همراه گزینه حذف انتخابی صدا برای کاهش چشمگیرتر حجم فایل خروجی.
- **📊 پیش‌نمایش حجم تقریبی تصویر:** تخمین هوشمند حجم فایل خروجی پیش از شروع فشرده‌سازی.
- **📋 تاریخچه فشرده‌سازی (Session History):** نمایش و دسترسی سریع به فایل‌های فشرده شده در طول جلسه کاری.
- **🎨 طراحی جدید و چشم‌نواز (Premium Glassmorphism):** رابط کاربری با پس‌زمینه اورب‌های متحرک رنگی، افکت شیشه‌ای عمیق‌تر، انیمیشن‌های نرم و افکت بارش کاغذ رنگی (Confetti) هنگام موفقیت.

## 🚀 راهنمای اجرای بسیار ساده به صورت محلی

### ۱. دریافت پروژه از گیت‌هاب
```bash
git clone https://github.com/AriaRazavi2005/offline-media-compressor.git
cd offline-media-compressor
```

### ۲. نصب وابستگی‌ها
فقط دستور زیر را اجرا کنید. فایل‌های WebAssembly موتور FFmpeg به صورت خودکار پس از نصب پکیج‌ها دانلود می‌شوند:
```bash
npm install
```

### ۳. اجرای سریع پروژه
کافیست دستور ساده زیر را بزنید تا برنامه در مرورگر شما باز شود:
```bash
npm start
```

## 🛠️ فناوری‌های استفاده شده

- **Core & Frontend:** React 19 + Vite
- **Video Engine:** `@ffmpeg/ffmpeg` + `@ffmpeg/core-mt` (Multi-threaded WebAssembly)
- **Image Engine:** HTML5 Canvas API
- **ZIP Creation:** JSZip
- **Styling:** Custom Modern Vanilla CSS (Glassmorphism & Neon Glows)
